Introduction

Web development is the process of creating websites and web applications. It is a broad term involving various technologies, including web design, web content management, web scripting, web server, and web browser. We can also refer to the web development process as the development of a website.This document contains a list of the most common web development technologies and tools.

Markup language that defines the structure and layout of web pages. CSS

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

WebStorage

Session Storage

Stores data for a particular “session”. If the tab or window is closed, the data is cleared. Storage size limit is almost 5MB

Local Storage

Canvas and WebGL

Used for animations, data visualization (charts, graphs, etc.), photo manipulation among other things. Canvas API in itself is limited to 2D graphics. WebGL (Web Graphics Library) is a JavaScript API used to create high performance, interactive 2D and 3D graphics without any plug-ins. Has the capability to use hardware graphics acceleration provided by the user's device.
